#  User Payment Management System

A desktop application built with WPF and MVVM architecture for efficient user and payment management.

##  Key Features
- **User Management**: Add, edit, and delete user profiles
- **Payment Tracking**: Record and manage payments per user
- **Data Persistence**: SQLite database with Entity Framework Core
- **Clean Architecture**: MVVM pattern with proper separation of concerns

##  Technologies
- C# 12
- .NET 8
- WPF / XAML
- Entity Framework Core 8
- SQLite
- MVVM Pattern
